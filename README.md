# 📊 SVG Data Visualizer

실시간 데이터를 기반으로 다양한 SVG 차트를 생성하고 커스터마이징할 수 있는 웹 기반 시각화 도구입니다. 복잡한 설정 없이 데이터를 업로드하고, 원하는 스타일과 애니메이션을 적용하여 전문가 수준의 차트를 만들 수 있습니다.

🔗 **라이브 데모**: [https://jvibeschool.org/DATASVG/](https://jvibeschool.org/DATASVG/)

![SVG Visualizer Splash](splash.jpg)

## ✨ 주요 기능

### 1. 📂 데이터 업로드 및 관리
- **다양한 형식 지원**: JSON, CSV 파일을 직접 드래그 앤 드롭하여 업로드할 수 있습니다.
- **샘플 데이터**: 매출, 인구, 기온, 주식 등 8가지 이상의 비즈니스 샘플 데이터를 제공합니다.
- **실시간 미리보기**: 업로드된 데이터를 테이블 형태로 즉시 확인하고 필드를 선택할 수 있습니다.

### 2. 📈 차트 생성 및 매핑
- **8가지 차트 유형**: 막대(Bar), 선(Line), 영역(Area), 파이(Pie), 도넛(Donut), 산점도(Scatter), 레이더(Radar), 가로 막대(Horizontal Bar).
- **스마트 매핑**: 데이터의 X축, Y축 필드를 자유롭게 선택하고 차트 제목 및 크기를 설정합니다.

### 3. 🎨 프리미엄 스타일 커스터마이징
- **컬러 팔레트**: Vibrant, Ocean, Sunset, Forest, Neon, Pastel 등 엄선된 6가지 테마.
- **다크/라이트 모드**: 사용자의 환경에 맞는 테마 토글 기능을 지원합니다.
- **세부 스타일**: 배경색, 투명도, 폰트(Outfit, JetBrains Mono), 테두리 곡률, 그림자 효과 등을 세밀하게 조정합니다.

### 4. 🎬 역동적인 애니메이션
- **다양한 효과**: FadeIn, SlideUp, Scale, Stagger, Drawing 등 5가지 이상의 애니메이션 유형.
- **세밀한 제어**: 지속 시간(Duration), 지연 시간(Delay), 이징(Easing) 함수를 설정하여 차트의 등장을 극대화합니다.
- **툴팁 및 인터랙션**: 마우스 호버 시 상세 데이터 표시 및 하이라이트 기능을 제공합니다.

### 5. 💾 다운로드 및 배포
- **SVG 내보내기**: 완성된 차트를 고해상도 SVG 파일로 즉시 다운로드하여 웹사이트나 문서에 바로 사용할 수 있습니다.

## 🛠️ 기술 스택

- **Core**: Vanilla JavaScript (ES6+), HTML5, CSS3
- **Visualization Library**: D3.js v7
- **Design**: CSS variables 기반의 Design System, Flexbox/Grid Layout
- **Typography**: Google Fonts (Outfit, JetBrains Mono)

## 🚀 시작하기

이 프로젝트는 별도의 서버 설정 없이 웹 브라우저에서 바로 실행할 수 있는 싱글 페이지 애플리케이션(SPA)입니다.

1. 이 저장소를 클론합니다:
   ```bash
   git clone https://github.com/your-username/svg-data-visualizer.git
   ```
2. `index.html` 파일을 브라우저로 엽니다.
